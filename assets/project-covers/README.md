# Project Cover Assets

Place custom project logo/cover images in this folder.

Recommended naming:
- `forest-rogue-like.png`
- `sell-keep-salvage.png`
- `tower-defense.png`
- `tailored-io.png`
- `nikkah-planner.png`
- `megabonk-ml.png`

The overview and in-depth cards will automatically use these files when present.
If a file is missing or fails to load, the UI falls back to a placeholder label.
